# 🚗 Road Accident Data Analysis Dashboard

A comprehensive **Microsoft Excel dashboard** designed to analyze road accident data and uncover meaningful insights into casualty severity, vehicle types, road conditions, lighting conditions, and accident locations. This interactive dashboard enables data-driven analysis to support road safety improvements and informed decision-making.

---

## 📖 Overview

This project presents an interactive **Road Accident Analysis Dashboard** built in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, and Data Visualization techniques. The dashboard transforms raw accident data into meaningful insights, helping users identify accident patterns, monitor casualty trends, and evaluate factors affecting road safety.

---

## 🎯 Project Objectives

- Analyze total road accident casualties.
- Compare accident severity (Fatal, Serious, Slight).
- Identify vehicle types with the highest casualties.
- Analyze accidents by road type and road surface.
- Compare accidents in Urban and Rural areas.
- Study the impact of light conditions on accidents.
- Analyze monthly casualty trends.
- Build an interactive dashboard for better decision-making.

- Slight severity accidents account for the largest share of casualties.

- Single carriageway roads experience the highest number of accidents.
- Urban areas report more casualties than rural areas.
- Most accidents occur during daylight conditions.
